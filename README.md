# tidyverse-intro
## Purpose
The purpose of this project is to provide an overview and introduction to R and
Tidyverse for institutional research professionals. This project began as a
webinar presentation to the Tennessee Association for Institutional Research
(TENNAIR) on December 4, 2025.
